Project: Home Server Revival

Background

I purchased a used MacBook on Craigslist for schoolwork, but its age and noise issues led me to rely on school computers instead. After discovering a Linus Tech Tips video on turning old computers into home servers, I decided to breathe new life into my MacBook.

Objective

I aimed to explore the possibilities of repurposing my old MacBook, even if it meant it might break and return to its dormant state. I also wanted to find a convenient way to access my extensive movie and book collections without cluttering my devices.

Methodology

Inspired by the LTT video, I opted to use Ubuntu-Server instead of Windows, leveraging my existing experience with Ubuntu and the Linux command line. I discovered Docker's potential while setting up a Jellyfin server and added more containers to the server. However, managing pages without Docker proved challenging, so I created an Apache page to provide easy access to all installed server elements.

Results

Through this project, I discovered the versatility of home servers and have continued to experiment with new features. My server now runs:

  • Jellyfin for movie management
  • Kavita Reader for ebook management
  • NextCloud for file storage and backup
  • Portainer for container management
  • A navigation page powered by Apache

This project has not only given my old MacBook a new purpose but also provided a centralized hub for my media and files, making it easier to access and manage my digital collections.

Discussion

When this project started, it was really just to see if I could make the movie server, but it has been a very fun experience and I have enjoyed working on it and adding features.
